VanillaGuide
============
Начинаю переводить на русский язык и делать актуализацию гайда по прокачке с 1 по 60 уровень для "Ванильной" версии игры WOW 1.12.1 EnGB
============
![Скрин](https://github.com/Eynschteyn/VanillaGuide/blob/main/Images/Info.png)

![Скрин](https://github.com/Eynschteyn/VanillaGuide/blob/main/Images/Guide.png)

В аддоне представлены:

* "Пошаговые" руководства по зонам на основе работы Джоаны, Брайана Коппса и RestedXP
* Подсказки и советы по различным квестам
* Интеграция: [MetaMapBWP](https://github.com/laytya/Metamap-vanilla)
* Интеграция: [TomTom Vanilla](https://github.com/cralor/TomTomVanilla)

Желательные Аддоны 
============
* Аддон для квестов: [pfQuest by Shagu](https://github.com/Eynschteyn/pfQuest)
* Аддон для гринда: [Shagukill by Shagu](https://github.com/Eynschteyn/ShaguKill)
* Аддон для интерфейса: [ShaguTweaks by Shagu](https://github.com/Eynschteyn/ShaguTweaks)
* Папка со шрифтами кирилицы: [Fonts - эту папку необходимо закинуть в корень игры WoW](https://github.com/Eynschteyn/Fonts)

Для работы аддона необходим шрифт с поддержкой кирилицы в противном случае кирилица не будет отображаться в аддоне.
============

> **ПРИМЕЧАНИЕ:**
> 
> - Данный аддон не нужно воспринимать как пошаговую инструкцию так как в прокачке очень много ньюансов и если вы играете без сна то будут моменты когда нужно будет набивать опыт занимаясь гриндом мобов :)
> - Все гайды были проверены на ядре сервера Nostalrius и Kronos.
